Como funciona?

Ao fazer o cadastro na plataforma o usuário tem a opção de criar um repositório novo e fazer o commit do código que está em desenvolvimento no repositório local para o repositório novo criado no Github. O Github vai armazenar o repositório e deixar que o usuário escolha deixar visível para outros ou não, ganhando destaque por sua rapidez, que faz com que os usuários consigam acompanhar o código em tempo real. Outro ponto importante é a ideia de rede social que o Github possui, possibilitando aos usuários compartilhar repositórios e até mesmo usar o repositório de outra pessoa para si.
